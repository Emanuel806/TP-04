Por: Emanuel Olivera

1. ¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

La etiqueta <table> es la utlizada para crear una tabla en HTML.

2. ¿Qué etiqueta se utiliza para definir una fila en una tabla?

La etiqueta <tr> (table row) es la utilizada para definir una fila en una tabla.

3. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

La etiqueta <th> (table heading) es la utilizada para crear celdas de encabezado en una tabla, distinguiéndolas de los demás datos.

4. ¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

La etiqueta <td> (table data) es la utilizada para definir la celda de una tabla que contiene datos.

5. ¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

El atributo colspan se utiliza para expandir una celda y que esta cubra varias columnas.

6. ¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

El atributo rowspan se utiliza para expandir una celda y que esta cubra varias filas.

7. Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

Para hacer eso, necesitaremos utilizar la etiqueta <tr> 10 veces, y llenarlas con contenido utilizando la etiqueta <td> si queremos visualizar 10 filas en la tabla.
